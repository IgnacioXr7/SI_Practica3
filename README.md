El objetivo principal es construir un modelo de aprendizaje automático capaz de analizar simultáneamente múltiples indicadores de salud para predecir con precisión el nivel de aptitud física de un individuo.

Objetivos específicos de la práctica:

    -Identificar Patrones Ocultos: Determinar qué variables (por ejemplo, el estrés o la calidad del sueño) tienen un peso real en la salud, más allá de los simples pasos diarios.

    -Automatizar la clasificación: Crear un algoritmo que, al recibir nuevos datos de un usuario, sea capaz de etiquetarlo correctamente como "Fit", "Moderate" o "Unfit" sin intervención humana.

    -Comparar tecnologías: Evaluar qué tipo de algoritmo (vecinos cercanos, redes neuronales o árboles de decisión) es más fiable para este tipo de datos de salud que tienen "reglas" complejas y umbrales críticos.

    -Asegurar la fiabilidad: Conseguir un modelo que no solo sea preciso con los datos que ya tenemos, sino que sea capaz de generalizar
